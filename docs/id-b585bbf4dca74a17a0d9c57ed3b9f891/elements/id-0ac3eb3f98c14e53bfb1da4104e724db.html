Article 11: Terminology ()
Common terminology practices shall ultimately contribute to reinforcing patient safety and increase the overall quality of the continuity of the care process. Guidelines should adopt the concept of "preferred code systems", for data elements where there is a consensus around current practices. Additionally, guidelines should also highlight strategic aims by pointing to upcoming code systems.
