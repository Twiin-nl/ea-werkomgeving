Zorgaanbieder authenticatie ()
VEREISTE - Zorgaanbieders moeten elkaars identiteit met zekerheid kunnen vaststellen. VERANTWOORDELIJKHEID - Dossierhouder en dossierraadpleger/ -ontvanger. INVULLING TWIIN - Identiteit van de zorgaanbieder: URA TOELICHTING - De communicerende zorgaanbieders dienen als identificatie het UZI-register Abonneenummer (URA) gebruiken. De authenticatie van deze identiteit kan nog niet (altijd) op een hoog niveau plaatsvinden, slechts het op Twiin aangesloten GtK kan geauthenticeerd worden op basis van een PKIo-servercertificaat.
