15.3 Promoting automated validation tools for interoperability ()
Promote the use of automated validation tools for technical and semantic interoperability criteria. 1. These tools should be extended to address the specificities of each type of data category being exchanged (e.g. data models, datasets, data formats, code systems, value sets). 2. These tools check and stress security requirements at infrastructure and application level. 3. These tools check automatically schema, schematron and model based validation
