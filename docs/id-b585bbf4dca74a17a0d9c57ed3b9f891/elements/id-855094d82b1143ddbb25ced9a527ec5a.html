12.4 Regular evaluation and versioning of the master catalogue and value set catalogues ()
The master catalogue should be evaluated on a regular basis with regards to the selection of concepts and the code systems used. For historical health data meaning preservation, Value set Catalogues should maintain previous versions of the code systems.
