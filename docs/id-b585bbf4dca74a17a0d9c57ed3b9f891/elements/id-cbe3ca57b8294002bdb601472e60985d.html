Inzage logging door patiënt ()
VEREISTE - De communicerende partijen moeten voorzieningen in stand houden waarmee inzage in de loggingbestanden tot op gebruikersniveau voor de betrokken patiënten mogelijk is. VERANTWOORDELIJKHEID - Dossierhouder en dossierraadpleger/-ontvanger INVULLING TWIIN - Deelnemers binden zich via de Voorwaarden Twiin Deelnemer aan het geven van inzage aan patiënten TOELICHTING - Wenselijk is een gemeenschappelijke voorziening waarmee de patiënt inzage krijgt in de logging (van de uitwisselingen) van deelnemers van Twiin, maar deze is nog niet beschikbaar.
