5.010 Systeem identificatie ()
OMSCHRIJVING - Om zich te kunnen authentiseren, kunnen alle systemen betrokken bij transacties in het kader van Twiin een geldig PKI-certificaat overleggen. DOMEIN OPMERKING - Een geldig PKI-certificaat is een UZI-servercertifcaat of een PKIoverheid-certficaat.
